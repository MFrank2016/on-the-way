# ⚡ 快速开始指南

## 🎯 5分钟快速上手

### 第一步：启动后端

打开终端，执行：

```bash
cd backend
go run main.go
```

看到这个输出就成功了：
```
Server is running on port 8080
```

### 第二步：启动前端

**新开一个终端窗口**，执行：

```bash
cd frontend
npm install   # 首次运行需要安装依赖
npm run dev
```

看到这个输出就成功了：
```
  ▲ Next.js 14.x.x
  - Local:        http://localhost:3000
```

### 第三步：开始使用

1. 打开浏览器访问：`http://localhost:3000`
2. 点击"立即注册"创建账号
3. 填写用户名、邮箱和密码（至少6位）
4. 登录后即可使用所有功能

## 🎮 功能体验

### 1. 任务管理
- 点击左侧"今日待办"
- 点击"添加任务..."
- 输入任务标题，回车即可

### 2. 番茄时钟
- 点击左侧"番茄专注"
- 选择时长（15/25/30/45分钟）
- 点击"开始专注"

### 3. 四象限
- 点击左侧"四象限"
- 查看任务的重要性和紧急程度
- 点击任务下方的优先级按钮切换

### 4. 统计分析
- 点击左侧"统计"
- 查看你的任务完成情况和专注数据

### 5. 习惯打卡
- 点击左侧"习惯打卡"
- 点击"添加习惯"按钮（功能正在开发）
- 每天点击"打卡"按钮

### 6. 倒数日
- 点击左侧"倒数日"
- 点击"添加倒数日"按钮（功能正在开发）
- 查看重要日期的倒计时

### 7. 搜索
- 点击左侧搜索框
- 输入关键词搜索任务、清单或标签

## ⚠️ 常见问题

### Q: 后端启动失败？
A: 确保已安装Go 1.21+，运行 `go version` 检查

### Q: 前端启动失败？
A: 确保已安装Node.js 18+，运行 `node -v` 检查

### Q: 无法注册用户？
A: 检查后端是否正常运行在8080端口

### Q: 登录后空白？
A: 查看浏览器控制台是否有错误信息

### Q: CORS错误？
A: 确保前端访问的是 `localhost:3000`，后端是 `localhost:8080`

## 🔧 环境检查

运行前请确保：

```bash
# 检查Go版本
go version
# 应该显示: go version go1.21.x 或更高

# 检查Node.js版本
node -v
# 应该显示: v18.x.x 或更高

# 检查npm版本
npm -v
# 应该显示: 9.x.x 或更高
```

## 💡 使用技巧

1. **快速添加任务**
   - 在今日待办或收集箱页面
   - 点击快速添加框
   - 输入标题后直接按回车

2. **完成任务**
   - 点击任务左侧的复选框
   - 任务会被标记为已完成
   - 可在列表中看到划线效果

3. **专注工作**
   - 使用番茄时钟保持专注
   - 推荐25分钟工作，5分钟休息
   - 完成4个番茄后，休息15-30分钟

4. **时间管理**
   - 使用四象限管理任务优先级
   - 重要且紧急的事情立即处理
   - 重要不紧急的事情提前规划

5. **养成习惯**
   - 设置简单易行的习惯
   - 每天坚持打卡
   - 连续21天习惯就会形成
