## 🚀 如何使用

### 快速启动

```bash
# 1. 启动后端
cd backend
go run main.go

# 2. 启动前端（新终端）
cd frontend
npm install
npm run dev

# 3. 访问应用
# 打开浏览器访问 http://localhost:3000
```

### 首次使用

1. 访问 `http://localhost:3000`
2. 点击"立即注册"创建账号
3. 登录后即可使用所有功能

### 功能体验

- **任务管理**: 在"今天"或"收集箱"添加任务
- **番茄专注**: 进入"番茄专注"页面开始计时
- **四象限**: 在"四象限"页面管理任务优先级
- **统计分析**: 查看"统计"页面了解你的数据
- **习惯打卡**: 在"习惯打卡"添加习惯并打卡
- **倒数日**: 在"倒数日"添加重要日期
- **搜索**: 使用侧边栏搜索框快速查找
